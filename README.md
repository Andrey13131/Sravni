
### 🛠 Памятка: Утилита Analyzer (Поиск потерь данных)

Привет! Это небольшая локальная программа, которая поможет вам мгновенно находить «потерянные» строки, артикулы или логи при сравнении двух списков. 

**Как это работает (3 простых шага):**
1. **Окно 1 (Основная таблица):** Вставьте сюда исходный (эталонный) список.
2. **Окно 2 (Вторая таблица):** Вставьте сюда список, который нужно проверить.
3. Нажмите синюю кнопку **«Найти потери»**. 
В нижнем окне моментально появятся все строки, которые есть в первом списке, но отсутствуют во втором.

**💡 Фишки, которые сэкономят вам время:**
* **Игнорирование регистра:** Программе всё равно, большие буквы или маленькие. `АКСП-123` и `аксп-123` она посчитает одинаковыми.
* **Умные счётчики:** Количество строк над окнами пересчитывается автоматически прямо во время вставки или удаления текста.
* **Быстрое копирование:** Не нужно выделять результат мышкой — просто нажмите кнопку **«Скопировать результат»**, и данные уже в буфере обмена.
* **Отмена действий:** Случайно удалили нужный текст? Стандартная комбинация **Ctrl+Z** вернёт всё на место. (Ctrl+C, Ctrl+V и Ctrl+A тоже работают идеально на любой раскладке).
* **Безопасность:** Программа полностью автономна и работает без интернета. Никакие корпоративные данные никуда не отправляются.

Для сброса всех окон перед новой проверкой просто нажмите **«Очистить всё»**. Успешной работы!
