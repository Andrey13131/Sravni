# 🔍 Sravni v5.0 | Data & SQL Analyzer

Универсальный инструмент для сотрудников техподдержки и аналитиков. Позволяет быстро сравнивать списки данных и подготавливать их для работы с базами данных (SQL).

---

### 🚀 Основные функции:

* **Поиск потерь:** Сравнивает два списка (Окно 1 и Окно 2) и выводит элементы, которые есть в первом, но отсутствуют во втором.
* **SQL Форматирование:** Новая кнопка **"Скопировать в SQL"** берет данные из первого окна и мгновенно превращает их в формат `'строка1', 'строка2'` для быстрой вставки в оператор `IN (...)`.
* **Умные счетчики:** Количество строк в каждом окне пересчитывается автоматически в реальном времени.
* **Защита от ошибок:** Программа игнорирует регистр (АКСП-123 и аксп-123 — это одно и то же) и лишние пробелы.
* **Горячие клавиши:** Поддержка Ctrl+C, Ctrl+V, Ctrl+A и Ctrl+Z работает корректно при любой раскладке клавиатуры.

---

### 🛠 Как пользоваться:

1.  **Для сравнения:** Вставьте эталон в Окно 1, проверяемый список в Окно 2 и нажмите **"Найти потери"**.
2.  **Для SQL:** Вставьте список артикулов/логов в Окно 1 и нажмите **"Скопировать в SQL"**. Теперь просто вставьте результат в ваш SQL-запрос.
3.  **Для очистки:** Нажмите **"Очистить всё"**, чтобы подготовить программу к новой задаче.

---

### 📦 Установка и запуск:

Если вы скачали исходный код (`Sravni.py`):
1. Установите Python 3.x.
2. Запустите файл через терминал: `python Sravni.py`.

Если вы используете готовую сборку:
1. Просто запустите `Analyzer.exe`. Программа полностью автономна и не требует установки.
